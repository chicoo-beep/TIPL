# Tankaria Breakout

An Android-friendly Arkanoid style game with Tankaria branding.

## Play

Open `index.html` in a browser.

## Arcade Features

- Missile, electric ball, fireball, magnet, freeze, gravity, mystery, and revive power-ups
- Bomb, rainbow, mystery, and power-up bricks
- Combo score multiplier for clean streaks
- Boss fights every 10 levels
- Touch-first controls for Android browsers

## GitHub Pages

This project is static and can be hosted directly with GitHub Pages from the repository root.
